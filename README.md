# Group Project

## Please, before adding a change:
1. `git pull master`
2. `git merge master`
3. `git status`
4. git `add <files>` or `git add .`

### Creating a branch
`git checkout -b <name of branch>`

### Switching to a branch
`git checkout <name of branch>`

## Add changes to the branch
`git push origin <name of branch>`
